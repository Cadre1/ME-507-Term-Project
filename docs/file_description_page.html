File Descriptions

Brief file descriptions:

  • main.c: The main program to be ran on the STM32 Microcontroller. Manages the multitasking of the two tasks as described in the Finite State Machines section and contains a debugging mode for manual control.
  • task1.c: The structure containing the methods and variables for task 1 as described in the Finite State Machines section.
  • task2.c: The structure containing the methods and variables for task 2 as described in the Finite State Machines section.
  • controller.c: The structure containing the methods and variables for closed-loop control as described in the Closed Loop Control Systems section.
  • motor_driver.c: A driver for the brushed DC motor.
  • servo_driver.c: A driver for the servo motor.
  • bno055.c: A driver for the 9 DOF IMU generated by Daniel Mironow at https://github.com/d-mironov/Bosch-BNO055-STM32.
  • encoder_driver.c: A driver for the integrated encoder of the brushed DC motor.
  • photoresistor_driver.c: A driver for the four photoresistors.
  • radio_reciever_driver.c: A driver for the radio receiver
  • vector.h: A supporting structure to handle vector notation.
  • intertask_vars.h: A supporting structure to handle intertask variables.
