Detailed Description

Initialize the Bosch BNO055 Sensor

Initializes the Bosch BNO055 Sensor with

the specified operation mode.

Args:

‍imu[bno055_t*]: pointer to the structure to initialize

------------------------------------------------------------—

Return:

‍bool: true if none of the init steps fail, false else
